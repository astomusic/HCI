#HCI final-Term Project
===

### 제목 / 목표
```
Next Portal / 키넥트를 이용하여 지나가는 사람과 인터렉션하는 작품을 만들어본다.
```

### 특징 (동작순서 / 기능)
```
복도를 지나가는 유저를 인식
유저의 스켈렉톤을 인식
유저의 스켈렉톤을 따라서 유려한 반응(미정 - 깃털날리기 또는 깃털 붙이기?)
(extra)해당 작업을 다른 복도에 있는 대상과 같이 반응하게 
```
### 개발순서
```
배경화면을 선정 해당 물체구현
-힌색 또는 단색의 배경 NEXT 글자(깃털, particle로 이루어짐)
-파티클의 구현과 배치에 대한 스터디 필요

키넥트로 사람을 인식 및 화면에 구현
-사람이 인식될 경우 사람의 head, hand, foot등의 node정보를 받아서 simple 사람형상을 구현한다.
-노드 정보를 get하고 얻어낸 정보로 부터 사람형상을 그리는 메소드 구현 필요

인식한 스켈렉톤에 대한 배경 물체와의 충돌 및 반응 구현
-해당 스켈렉톤 노드로 부터 NEXT글자의 충동 체크 구현
-충돌할 경우 깃털, particle이 날아감
-서서히 돌아오며, 충돌할 대상이 없는 경우 원래 위치에 안착
-파티클의 자연스러운 움직임에 대한 스터디 필요

다른 장소의 2명의 유저일 경우에 대한 충돌과 반응 구현
-위와 같은 원리로 반응하며 다른 키노트로 인식된 스켈렌톤 객테의 경우 다른 색을 부여 하여 구분
-네트워크 방법을 결정하고, 해당 네트워크로 스켈렉톤의 노드 정보를 주고 받는다.
-네크워크 방법과 구현에 대한 스터디 필요
```
